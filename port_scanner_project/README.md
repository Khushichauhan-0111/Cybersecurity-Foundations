# Network Port Scanner

## Description
A lightweight Python-based tool designed to perform network reconnaissance by scanning a target IP address to identify open ports. This project serves as a foundational tool for understanding network connectivity and service discovery.

## Key Features
- Scans user-defined port ranges.
- Uses Python's `socket` library for efficient network communication.
- Implements basic error handling (e.g., handling connection timeouts and hostname resolution).

## How to Run
1. Ensure Python 3.x is installed.
2. Run the script: `python port_scanner.py`

## Learning Objectives
- Understanding TCP/IP socket programming.
- Hands-on practice with networking fundamentals and service identification.
- Development of defensive tools for security auditing.
